  • Category: Data Analyst

  • Project date: 27 March, 2023

This is an portfolio detail

This project is Assignment 1 of the IoT with Smart Capabilities course. This project asks us to create a program that can filter datasets using the LPF and MVA methods as taught in previous modules. The following is the dataset used for filtering:

https://archive.ics.uci.edu/ml/datasets/Activity+Recognition+from+Single+Chest-Mounted+Accelerometer

After entering the dataset, we have to divide columns B, C, or D as the data used to filter. Then, create LPF and MVA programs using the specified Columns. Then, analyze the results of the LPF and MVA filters by displaying the data plots. Then, what is the FC that is used for the LPF, and how do I determine the FC, as well as the "window size" used in filtering?

I have included my GitHub as an additional attachment for more information about program results and report files that have been created. Please click "My Github" on the sidebar.
