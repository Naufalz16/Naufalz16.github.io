This is an portfolio detail

This project is a ProjectBased assignment from a machine learning course. This project-based work is carried out in groups with four members. The formulation of this task is to predict the level of vehicle fuel economy/MPG (miles per gallon: the average distance traveled by car in miles for each gallon of fuel consumed) based on a given car profile, which is represented by attributes such as cylinder, power ( horsepower), year of output, etc. The dataset can be retrieved from the following link:

https://drive.google.com/drive/folders/1HmKavcNCij76k02nCYQdP012cepM2nmz?usp=sharing

As for the tasks performed, namely carrying out all the exploration techniques and data pre-processing, which needs to be done, then it is explained why it is necessary to do this technique and analyze the results. Then, the model will be built using the pre-processed data, and a machine-learning process will be performed to get the best results. Then, please select the appropriate evaluation method and its justification. Then, various experiments involving exploration and pre-data processing, modeling, and evaluation stages will be carried out to get the best results and provide conclusions from all the processes that have been carried out and the final results of the experiments that have been carried out.
