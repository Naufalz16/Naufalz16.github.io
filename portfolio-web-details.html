  • Project date: 3 October, 2022
  • Client: RPL Course Assignment Project

This is an portfolio detail

This project is a web-based cash register, where my teammates and I design a web that can become a cash register for a store or a growing business. Cashy Cashier is a modern web-based cashier that can make it easier for users, especially workers who work as cashiers in a shop or restaurant who need the Cashy Cashier application to facilitate their work. In addition, this application can also add details of products sold in the store, add and subtract items from the database, and view transaction history to see an overview of their profits while using the application. I am here acting as a front-end by creating several web views, such as the "Pendapatan" and "Stok Barang" sections, which will later be done for the back-end by another colleague. I also added several other details, such as creating a "Kasir" section and adding a button to the "Stok Barang" section.
